# Boat_Race
 Boat Racing Simulation game that shows OOP using classes, properties, and methods

 1. Select type of boat to race.
 2. Name your boat if you want.
 3. Select race course.
 4. Program will simulate boat races and show statistics and odds calculations for each boat victory.
 5. Select a wager amount (use stats to decide how much to risk).
 6. Select who you think will win.  Use sim statistics to make decision.
 7. Results will be displayed and winnings/losses will be calculated and displayed.
